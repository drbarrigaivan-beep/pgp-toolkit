# PGP Toolkit

Herramienta simple para generar una nota técnica preliminar de PGP a partir de datos básicos.

## Funcionalidades
- Carga de afiliados y servicios (CSV)
- Indicadores automáticos
- Estimación preliminar de PGP
- Descarga de nota técnica

## Aviso
Esta herramienta genera una estimación preliminar y no reemplaza validación actuarial, legal o financiera.
